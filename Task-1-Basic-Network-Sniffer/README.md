# 🛡️ Basic Network Sniffer

## 📌 Overview

This project was completed as part of my **CodeAlpha Cyber Security Internship**.

The objective of this project was to develop a basic network packet sniffer using Python. The program captures live network packets and displays important information such as source and destination IP addresses, protocol type, port numbers, packet number, and timestamp.

---

## 🎯 Objectives

- Understand how network packets travel across a network.
- Capture live network traffic.
- Analyze packet information.
- Learn the basics of network monitoring.
- Gain practical experience with Python networking.

---

## 🚀 Features

- ✅ Captures live network packets
- ✅ Displays packet number
- ✅ Shows capture time (timestamp)
- ✅ Displays Source IP Address
- ✅ Displays Destination IP Address
- ✅ Detects Protocol (TCP, UDP, ICMP)
- ✅ Displays Source Port
- ✅ Displays Destination Port
- ✅ Clean and readable output

## ▶️ How to Run

### 1. Install Scapy

```bash
pip install scapy
```

### 2. Install Npcap

Download and install Npcap from:

https://npcap.com

### 3. Run the program

```bash
python network_sniffer.py
```
